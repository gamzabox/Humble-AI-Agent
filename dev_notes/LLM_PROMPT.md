# 채팅 히스토리 및 취소 기능 구현 Prompt
현재까지 구현된 채팅 프로그램에 채팅 히스토리 기능과 취소 기능 추가해줘
- 개발언어는 golang을 사용할 것
- UI 는 fyne 를 이용할 것
- 전체 요구사항은 REQUIREMENTS.md 파일에서 확인 하고 필요시 내용 추가 할것
- Task 를 TASKS.md 파일의 Chat History and Canceling Tasks 항목에 정리하고 완료한 Task 를 체크 표시 할 것
- 반드시 테스트 코드를 먼저 작성한 다음 기능을 구현하고 테스트를 수행해 PASS 할 경우 다음 Task 를 진행 할 것
- 화면 레이아웃을 죄우로 나누고, 좌측엔 저장된 채팅 타이틀 리스트를 우측엔 채팅 화면로 구성
- 죄측 화면의 채팅 타이틀을 선택하면 저장된 채팅 기록을 우특 화면에 보여줘야함 
- 채팅을 입력하고 Send 버튼을 누르면 inputbox 는 비활성화 되고 Send 버튼은 Cancel 버튼으로 변경되어야 함
- 채팅에 대한 답변을 모두 받은 다음에 inputbox 는 다시 활성화 되고 Cancel 버튼도 Send 버튼으로 돌아 와야 함

# 채팅 세션 관련 수정 사항
새로운 세션 만들기 기능 추가 해줘
- 앱 실행시 항상 새로운 세션에서 시작
- 좌측 패널 최 상단에 "새 새션" 버튼을 추가하고 이 버튼을 누르면 새로운 세션 시작
- 패널 Split 비율을 좌측 30% 우측 70% 로 조정하고 사용자가 조절하는 기능은 지금 대로 유지 할것
